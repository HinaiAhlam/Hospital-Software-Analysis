Patient-Facing User Stories



Appointment Booking



As a patient, I want to book an appointment online so that I can avoid waiting in queues.



Medical Records Access



As a patient, I want to view my medical records online so that I can track my health history.



Prescription Management



As a patient, I want to receive digital prescriptions so that I can easily access my medications.



Bill Payment



As a patient, I want to pay my bills online so that I don’t have to wait at the billing counter.



Notifications



As a patient, I want to receive reminders for appointments and tests so that I don’t miss them.

. Doctor-Facing User Stories



Patient History Access



As a doctor, I want to access a patient’s medical history so that I can make informed decisions.



Test Results Review



As a doctor, I want to receive lab test results digitally so that I can provide faster diagnosis.



Prescription Writing



As a doctor, I want to generate e-prescriptions so that patients can get medicines without errors.



Appointment Management



As a doctor, I want to view my daily appointments so that I can manage my time efficiently.



Consultation Notes



As a doctor, I want to record patient consultation notes digitally so that records are accurate and retrievable.

Nurse-Facing User Stories



Patient Monitoring



As a nurse, I want to update patient vitals in the system so that doctors have real-time information.



Medication Administration



As a nurse, I want to receive notifications for scheduled medications so that patients get timely doses.



Shift Handover



As a nurse, I want to review patient logs from previous shifts so that I understand ongoing care needs.



Patient Alerts



As a nurse, I want to receive alerts for emergency situations so that I can respond quickly.



Lab Technician User Stories



Test Requests Management



As a lab technician, I want to view all pending test requests so that I can process them efficiently.



Result Uploading



As a lab technician, I want to upload test results directly into the system so that doctors can access them immediately.



Inventory Check



As a lab technician, I want to check the availability of lab supplies so that I can avoid delays in tests.

Administrative User Stories



Patient Registration



As an admin, I want to register new patients quickly so that the hospital workflow is smooth.



Billing \& Insurance



As an admin, I want to generate invoices and process insurance claims so that billing is accurate.



Staff Scheduling



As an admin, I want to create and manage staff schedules so that there is adequate coverage at all times.



Reporting \& Analytics



As an admin, I want to generate reports on hospital operations so that management decisions are data-driven.
